# 🛰️ AstroSafra - Dados Espaciais para o Pequeno Agricultor

> Solução Agritech para democratizar o acesso à inteligência de satélite, protegendo colheitas e garantindo a segurança alimentar.

## 📝 O Problema
Pequenos produtores perdem bilhões de reais por ano devido a mudanças climáticas e pragas. Enquanto o grande agronegócio utiliza satélites e IA, 77% dos estabelecimentos familiares não têm acesso a essa tecnologia, ameaçando o abastecimento local (ODS 2 - Fome Zero).

## 💡 A Solução
O AstroSafra atua como um tradutor tecnológico:
* Conecta-se a APIs públicas de satélites (Copernicus/NASA).
* Uma IA traduz os dados brutos (infravermelho, umidade) em recomendações simples enviadas para o celular do agricultor (ex: "Irrigar a área leste hoje").
