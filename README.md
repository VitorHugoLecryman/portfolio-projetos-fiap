## 🛠️ Tecnologias Dominadas e Praticadas

Através destes projetos, foram consolidadas habilidades práticas nas seguintes frentes:

* **Desenvolvimento Mobile:** Desenvolvimento nativo para Android utilizando Kotlin e arquitetura moderna.
* **Banco de Dados:** Modelagem lógica e física de dados relacionais, criação de scripts SQL robustos, automação com PL/SQL, Triggers e Procedures (padrão Oracle).
* **Design de Interface (UI/UX):** Prototipagem de alta fidelidade no Figma, fluxos de navegação, arquitetura de informação e usabilidade.
* **Visão de Negócio & Inovação:** Modelagem de modelos de receita, análise de mercado, estruturação de propostas de valor e apresentações de Pitch.
